 1) HTML:
 
 <p> - блочный тег, он занимает всю доступную ширину
 <b> - строчный тег, он занимает только часть

Нужно блочные теги можно вкладывать только в блочные, а строчные куда угодно.

<h1> - главный заголовок страницы
<button> - для кнопки

Если не знаем какой тег использовать:
<div> - блочный тег (занимает всю ширину)
<span> - строчный тег (занимает только контент)

1. Теги для заголовков и параграфов:
Для заголовков - <h1> ... <h6>
Для параграфов - <p>
Для списка (не упорядоченного маркир.) - <ul> ... <li>
Для списка (упорядоченного маркир.) - <ol> ... <li>

2. Теги для форматирования текста:
<b> и <strong> - жирный текст
<i> и <em> - для курсива
<s> - перечеркнутый текст
<u> - подчеркнутый текст
<sub> - подстрочный текст (формулы)
<sup> - надстрочный текст (степени)
<small> - маленький шрифт
<mark> - выделение желтого цвета
<br/> - перенос строки 
<hr/> - горизонтальная линия
<code> - для кода
<pre> - удаление пробелов и переноса строк

3. Теги для ссылок и кнопок:
<a href="...">...</a> - ссылка
<a href="#...">...</a> - якорная ссылка (значение после # это атрибут id)
<a href="..." download>...</a> - ссылка на скачивание файла
<a href="mailto:...">...</a> - ссылка для отправки письма
<a href="tel:+799999">...</a> - ссылка на телефон
<a href="..." target="_blank">...</a> - ссылка открывается в новом окне
<button type="...">...</button> - кнопка 
в type пишем:
	submit - для отправки данных на сервер
	reset - сброс введенных значений
	type="button" - для других кнопок
